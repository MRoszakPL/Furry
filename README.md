# Furry
Simple game created during Coder's Lab bootcamp

#### Main objective of the game

Eat coins to increase your score. How many can you eat?

### Used technologies

- HTML
- CSS
- JavaScript
- Webpack

### How to run?

Just open index.html in your browser
